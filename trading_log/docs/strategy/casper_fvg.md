# 캐스퍼 (Casper) SMC FVG (Fair Value Gap) 매매 전략

이 문서는 AI 트레이딩 봇이 특정 시간대(단기 모멘텀)에 유동성 갭(FVG)을 활용해 매매 타점을 잡는 **"캐스퍼의 90분 FVG 단타 전략"**을 분석기에 적용하기 위해 정리한 기준 문서입니다.

## 📌 핵심 철학
- **짧은 시간, 높은 효율**: 하루 종일 차트를 보며 뇌동매매를 하는 대신, 유동성이 가장 풍부한 **특정 90분(예: 뉴욕장 오픈 직후)**에만 매매하여 승률을 극대화합니다.
- **FVG (Fair Value Gap, 공정 가치 갭)**: 가격이 급등/급락하며 매수·매도 거래가 치열하게 이뤄지지 못하고 '비어버린' 캔들 사이의 공간. 시장은 이 불균형을 채우기 위해 십중팔구 해당 가격대로 **'되돌림(회귀)'** 현상을 보입니다.
- **아무 FVG나 들어가지 않는다**: 시간, 기준선 돌파, FVG 형성이라는 **3가지 필수 조건**이 모두 충족된 곳에서만 진입합니다.

---

## 🔍 FVG (Fair Value Gap) 식별 방법
차트상 연속된 3개의 캔들을 봅니다.
1. **상승 FVG (롱 타점)**: 1번 캔들의 고가와 3번 캔들의 저가 사이에 겹치는 꼬리가 없이 뻥 뚫린 빈 공간 (2번 캔들의 거대한 양봉 몸통 구간).
2. **하락 FVG (숏 타점)**: 1번 캔들의 저가와 3번 캔들의 고가 사이에 겹치는 꼬리가 없는 빈 공간 (2번 캔들의 거대한 음봉 몸통 구간).

---

## 🟢 롱(매수) 포지션 진입 전략 (3가지 필수 조건)

### 1단계: 유동성이 몰리는 '특정 시간' 지정
- 24시간 매매를 금지합니다.
- KIS API(한국 주식) 적용 시 **오전 9시 00분 ~ 10시 30분 (장 초반 90분)** 같이 매수세가 폭발하는 시간대로 한정합니다. (미장 기준은 뉴욕장 오픈인 밤 10시 30분 또는 11시 30분)

### 2단계: 상위 타임프레임(15분/5분봉) '기준선' 드로잉
- 매매할 타임프레임보다 한 단계 높은 캔들을 봅니다. (예: 5분봉 단타 시 15분봉 확인 / 1분봉 단타 시 5분봉 확인)
- 시작 시간의 첫 번째 상위 캔들(예: 9시 정각의 15분봉 캔들)의 **고가(High)와 저가(Low)**에 각각 가로 기준선을 긋습니다.

### 3단계: 기준선 돌파 및 FVG 생성 확인 (타점)
- 이제 하위 타임프레임(예: 5분봉 또는 1분봉) 차트로 내려옵니다.
- **조건**: 캔들이 2단계에서 그어둔 **'고가 기준선'을 몸통으로 강력하게 상향 돌파**하면서, 그 자리에 **상승 FVG**가 만들어지는지 관찰합니다.

### 💸 매매 실행 (진입/손절/익절)
- **진입 타점 (되돌림 매수)**: 주가가 급등 후 잠시 빠지면서, 앞서 만들어진 **'FVG 빈 공간' 안으로 가격이 다시 진입(되돌림)할 때** 지정가 매수를 들어갑니다.
- **손절가 (Stop Loss)**: 해당 FVG를 만들어낸 직전 캔들의 저점 밑으로 이탈 시 손절.
- **익절가 (Take Profit)**: 손익비 **1 : 2** 또는 **1 : 3** 으로 세팅.

---

## 🔴 숏(매도) 포지션 진입 전략
롱 전략을 반대로 뒤집습니다. (한국 주식 현물은 공매도가 어려우나, 이 시그널을 '기존 주식 전량 매도 및 관망' 필터로 사용)

1. **지정 시간**: 장 초반 90분
2. **기준선**: 첫 상위 캔들(15분봉)의 저가(Low) 선에 주목.
3. **돌파 및 FVG**: 하위 캔들이 **'저가 기준선'을 음봉 몸통으로 깨고 하락**하며, 그 자리에 **하락 FVG**가 발생.
4. **진입**: 주가가 살짝 반등하여 **하락 FVG 공간 안으로 되돌림을 줄 때** 숏(매도) 진입.
5. **손절가**: FVG 생성 캔들 직전 고점 돌파 시 손절 / **익절가**: 손익비 1:2.

## 🤖 AI 봇 적용 방안 (Next Step & 한계점)
앞으로 `quant_analyzer.py` 모듈 쪽에 이 SMC 로직을 추가하려면 다음의 기법적 설계가 필요합니다.

1. **시간 제한 모듈 (Time Window)**
   - `ai_trading_bot.py`루프에서 현재 시간이 09:00 ~ 10:30 사이가 아닐 경우, "FVG 전략 휴식(관망)" 모드로 전환하는 타임 락커(Time Locker) 구현.
2. **멀티 타임프레임 데이터 호출**
   - KIS API를 통해 단순 1일봉(Daily)이 아닌, **15분봉과 5분봉 데이터**를 동시에 가져와야 합니다.
   - 첫 15분봉의 High/Low 값을 변수에 저장(`target_high`, `target_low`).
3. **배열 3개로 FVG 검출 로직 작성**
   - 판다스 데이터프레임에서 연속된 인덱스 `i-2, i-1, i` 캔들을 읽어들입니다.
   - 롱 갭: `High[i-2] < Low[i]`가 True인지 검사하여 갭 박스의 상단(`Low[i]`)과 하단(`High[i-2]`) 좌표를 기록.
4. **자동 되돌림 감지 알림**
   - 현재가가 기록해 둔 갭 박스 범위 안으로 들어오면 텔레그램 봇으로 **"🚨 [캐스퍼 전략] 종목 X 상승 FVG 되돌림 도달! 매수 타점!"** 알림을 전송.
